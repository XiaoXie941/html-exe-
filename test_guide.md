# 网页打包工具测试指南

## 快速测试

### 方法1：使用示例文件测试

1. 运行工具：
   ```bash
   python run.py
   ```

2. 在工具界面中：
   - 选择"本地HTML文件"模式
   - 点击"浏览..."选择 `example.html`
   - 点击"预览"查看效果
   - 配置窗口参数（可选）
   - 点击"开始打包"

### 方法2：使用网页URL测试

1. 运行工具
2. 选择"网页URL"模式
3. 输入测试网址：`https://www.baidu.com`
4. 点击"预览"查看效果
5. 点击"开始打包"

### 方法3：使用文件夹测试

1. 创建一个测试文件夹，放入HTML文件
2. 运行工具
3. 选择"本地文件夹"模式
4. 选择测试文件夹
5. 工具会自动显示文件夹结构
6. 点击"开始打包"

## 故障排除

### 问题：工具无法启动
- 检查Python安装：`python --version`
- 检查依赖：`pip list | findstr "webview"`
- 重新安装依赖：`pip install -r requirements.txt`

### 问题：打包失败
- 检查网络连接
- 检查输出目录权限
- 查看详细错误日志

### 问题：生成的exe无法运行
- 检查目标系统是否满足要求
- 检查是否安装了WebView2运行时
- 尝试在其他Windows电脑上测试
